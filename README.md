# Heart & Lung Sound Classification System

## 📌 Description
This project detects whether heart and lung sounds are normal or abnormal using a CNN deep learning model.

## 🚀 Features
- Upload audio (.wav)
- Predict Normal / Abnormal
- Generate waveform & spectrogram
- Download PDF report
- View analytics dashboard

## 🧠 Model Used
- CNN (Convolutional Neural Network)

## 🛠️ Technologies
- Python
- Flask
- TensorFlow
- Librosa
- Matplotlib

## ▶️ How to Run
pip install -r requirements.txt
python app.py
